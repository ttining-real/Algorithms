## Day 1-2. 몫 구하기

### 📜 문제 설명

정수 `num1`, `num2`가 매개변수로 주어질 때,
`num1`을 `num2`로 나눈 몫을 `return` 하도록
`solution` 함수를 완성해주세요.

<br>

### ⚠️ 제한 사항

```javascript
0 < num1 <= 100;
0 < num2 <= 100;
```

<br>

### 👀 입출력 예

| `num1 ` | `num2` | `result` |
| ------- | ------ | -------- |
| 10      | 5      | 2        |
| 7       | 2      | 3        |
